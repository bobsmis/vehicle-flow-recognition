总体数据流

主流程由成员 5 串联其他成员模块。

detector = MotionDetector(params, roi_points, frame.shape)
tracker = CentroidTracker(params)
counter = LineCounter(roi_points, line_points)

detections, fg_mask = detector.detect(frame)
active_tracks = tracker.update(detections)
events = counter.update(active_tracks, frame_index, timestamp_sec)
counts = counter.counts()
annotated = draw_overlay(frame, roi_points, line_points, detections, active_tracks, counts, events)

修改接口检查清单

公开接口、共享模型字段、坐标格式、边界框格式、counts() 的 "total" 键发生变化时，需要同步本文档和对应调用方。
  • 是否改变了公开函数或类的参数？
  • 是否改变了 Detection、Track、CountEvent、ProcessingState 的字段？
  • 是否改变了 counts() 的键名，尤其是 "total"？
  • 是否改变了帧、掩膜、坐标或边界框格式？
  • 是否直接调用了其他成员文件中以下划线开头的私有函数？
  • 是否同步更新了本文档、README 和对应调用方？
