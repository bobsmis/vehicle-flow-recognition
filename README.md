# 车流量统计系统

这是一个基于 Python + OpenCV 的车流量统计 GUI 项目。项目不使用 YOLO、DNN 或任何深度学习检测模型，核心流程为：MOG2 高斯混合背景建模、形态学去噪、轮廓几何筛选、质心轨迹跟踪、虚拟计数线穿越统计。

## 运行环境

- Python 3.10 或更高版本
- OpenCV、NumPy、Pillow

安装依赖：

```powershell
python -m pip install -r requirements.txt
```

启动：

```powershell
.\run.bat
```

如果 `run.bat` 提示找不到 Python，请安装 Python 并把它加入 PATH。

## GUI 使用流程

1. 点击“选择视频”，导入任意本地测试视频。
2. 在首帧画面上选择“绘制 ROI 多边形”，依次点击道路检测区域的边界点。
3. 选择“绘制虚拟计数线”，点击两个端点生成计数线。
4. 根据视频调整检测参数；默认参数适合白天固定机位的车辆运动检测。
5. 可填写人工总数真值，用于导出准确率和误差。
6. 点击“开始”处理视频，可暂停、继续或停止。
7. 处理完成后，在输出目录中查看标注视频、过线记录 CSV、汇总 CSV 和 JSON。

## 输出文件

每次处理会在输出目录下生成一个时间戳子目录：

- `*_annotated.mp4`：带 ROI、计数线、目标框、轨迹和计数结果的标注视频。
- `过线记录.csv`：按过线顺序记录“过线第几辆车”和“过线时间”。
- `统计汇总.csv`：总数、人工真值、误差和准确率汇总，字段名为中文。
- `统计汇总.json`：与汇总 CSV 等价的中文结构化结果。

处理运行时，画面下方的“实时统计”会持续显示进度、总数、当前目标和处理 FPS；标注视频左上角也会叠加中文统计结果。

## 参数说明

- 背景历史、背景阈值、学习率：控制 MOG2 背景建模灵敏度。
- 模糊核、形态核：控制前景去噪和目标区域连接。
- 最小/最大面积：过滤过小噪声和过大异常区域。
- 矩形度、圆形度、宽高比：基于外接矩形和轮廓形状筛选车辆候选。
- 匹配距离、丢失帧数、轨迹长度：控制多目标关联和轨迹保留。

## 计数说明

系统按单行道总量统计车辆，目标任意方向穿过计数线都会计为 1 辆车；同一轨迹只会计数一次。

## 测试

不依赖 OpenCV 的计数逻辑测试：

```powershell
python -m unittest discover -s tests
```

安装 OpenCV 后，可以用下面脚本生成一个简单合成测试视频：

```powershell
python tests\create_synthetic_video.py
```

然后在 GUI 中导入 `tests\synthetic_traffic.mp4`，ROI 可覆盖整幅画面，计数线可画在画面中间。
